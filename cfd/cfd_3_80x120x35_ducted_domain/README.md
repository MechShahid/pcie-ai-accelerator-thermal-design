# CFD-3: 80 × 120 × 35 mm Ducted Heatsink

This folder contains figures and supporting files for the CFD-3 extended-length ducted heatsink simulation.

CFD-3 tested the 80 × 120 × 35 mm aluminium heatsink in a ducted air-domain configuration.

Main result:

- Maximum chip temperature: 83.52 °C
- Pressure drop: 49.02 Pa
- Result: Passed the 85 °C maximum chip-temperature target
