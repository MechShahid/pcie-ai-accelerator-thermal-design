# CFD-2: 80 × 100 × 35 mm Ducted Heatsink

This folder contains figures and supporting files for the CFD-2 ducted heatsink simulation.

CFD-2 tested the 80 × 100 × 35 mm aluminium heatsink in a ducted air-domain configuration.

Main result:

- Maximum chip temperature: 87.78 °C
- Pressure drop: 44.54 Pa
- Result: Improved compared with CFD-1, but still slightly above the 85 °C target

Detailed results are documented in:
